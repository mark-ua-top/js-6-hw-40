## ✨ Функціонал

- ✅ Реєстрація та авторизація (локально)
- ✅ Додавання/видалення контактів
- ✅ Пошук контактів за ім'ям

- **React 19** — UI бібліотека
- **Redux Toolkit** — управління станом
- **React Router** — маршрутизація
- **LocalStorage API** — збереження даних
- **Tailwind CSS** — стилізація
- **Vite** — збірка проекту

## 📦 Встановлення

```bash
# Клонування репозиторію
git clone https://github.com/mark-ua-top/js-6-hw-40.git
cd js-6-hw-40

# Встановлення залежностей
npm install
```

## 🖥️ Локальний запуск

```bash
# Запуск dev-сервера
npm run dev
```

Сервер запуститься на **http://localhost:3000**

## 🌐 Доступ з інших пристроїв у локальній мережі

При запуску `npm run dev` ви побачите щось на кшталт:

```
➜  Local:   http://localhost:3000/
➜  Network: http://192.168.1.100:3000/
```

**Network адреса** — це адреса для доступу з інших пристроїв у вашій Wi-Fi мережі (телефон, планшет, інший комп'ютер).

### 📱 Як підключити інші пристрої:

1. Переконайтесь, що всі пристрої підключені до **однієї Wi-Fi мережі**
2. На іншому пристрої відкрийте браузер
3. Введіть **Network адресу** (наприклад `http://192.168.1.100:3000`)
4. Готово! Застосунок працює

## 🔄 Синхронізація між пристроями

Оскільки дані зберігаються локально в кожному браузері, для синхронізації використовуйте **Експорт/Імпорт**:

### Експорт (з пристрою-джерела):
1. Увійдіть в акаунт
2. Натисніть кнопку **🌐** (біля кнопки "Вийти")
3. Виберіть **"Експорт даних"**
4. Збережіть JSON файл

### Імпорт (на новому пристрої):
1. Відкрийте застосунок
2. Натисніть кнопку **🌐**
3. Виберіть **"Імпорт даних"**
4. Виберіть збережений JSON файл
5. Перезавантажте сторінку
6. Тепер можна увійти з тими ж даними!

## 🔒 Де зберігаються дані?

Всі дані зберігаються в **localStorage** браузера:

| Ключ | Опис |
|------|------|
| `contacts_app_users` | Зареєстровані користувачі |
| `contacts_app_current_user` | Поточна сесія |
| `contacts_app_contacts` | Контакти (прив'язані до користувача) |

⚠️ **Важливо:** Якщо очистити дані браузера — дані втратяться!
